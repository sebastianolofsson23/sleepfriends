How to set up the Shortcut ▸
  1. Open the Shortcuts app → tap +
  2. Add: Find Health Samples → Type: Sleep Analysis, sort by Start Date newest first, limit 50, in the last 24 h
  3. Add: Filter items where Category Value is Asleep
  4. Add: Get First Item from filtered list → store as First Sample
  5. Add: Get Last Item from filtered list → store as Last Sample
  6. Add: Calculate Statistics → Sum of Duration → store as Total Minutes
  7. Add: Text action → paste the JSON body below (replace placeholders with your copied values and the Shortcut variables)
  8. Add: Get Contents of URL → URL: your Firestore URL above → Method: POST → Headers: Content-Type: application/json → Body: Text from step 7
  9. Turn it into an Automation → triggers when you turn off your morning alarm
